# gsync

Tool to sync between two folders on same computer using git. Usefull when need developed on WSL but execute on Windows

Tip: If need to have acces to folder in onedrive on WSL create a soft link on WSL

```
ln -s /mnt/c/path/on/windows ./NameOfFolderInWsl
```
*you can add the file NameOfFolderInWsl to .gitignore to be ignored*
